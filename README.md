# lorenzodeleo.com

Lorenzo de Leo's personal site: operator-investor positioning, speaking (AI-native organizations / startups / corporations), writing, bio, contact.

## Stack

Astro 5, TypeScript strict, Tailwind 4. Static output. Skeleton mirrored from `ldeleo/enya-website` (same primitives, own design tokens — warm paper / ink / terracotta, deliberately distinct from ENYA's violet).

## Development

```
npm install
npm run dev      # http://localhost:4321
npm run build
npm run preview
```
